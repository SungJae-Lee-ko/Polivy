❓ FAQ
================================================================================

Q1: build_exe.bat가 안 실행됨
A: 관리자 권한으로 CMD를 열고 다시 시도하세요.

Q2: Python을 찾을 수 없습니다
A: https://www.python.org에서 Python 3.9+ 설치 (PATH 추가 필수)

Q3: 빌드 후 exe가 없습니다
A: build와 dist 폴더를 삭제하고 build_exe.bat을 다시 실행하세요.
   rmdir /s /q build dist
   build_exe.bat

Q4: Google API Key 오류
A: .env 파일이 DC_Doc_Generator.exe와 같은 폴더에 있는지 확인하세요.
   GOOGLE_API_KEY=YOUR_API_KEY_HERE 형식으로 입력하세요.

Q5: 포트 8501이 이미 사용 중입니다
A: 기존 exe 프로세스를 종료하세요:
   taskkill /IM DC_Doc_Generator.exe /F
